# 🚀 DSA Fundamentals Repository

This repository contains all the **fundamental concepts and problems of Data Structures & Algorithms (DSA)**.

🎯 The goal is to build a strong base in DSA by practicing structured problems from **Arrays, Strings, Matrix, and Recursion**.
